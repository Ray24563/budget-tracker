
# WhyHub - Frontend Setup

This section is for setting up the frontend of WhyHub in your local computer.

### 1. Install dependencies
```bash
cd frontend
npm install
```

### 2. Create '.env' file
Inside the .env file, input the backend path
```bash
VITE_API_URL=(backend_server)
```

### 3. Run the Server
```bash
npm run dev
```

## 📌 Overview
**WhyHub** is a personal budget tracking web application designed to help in managing income and expenses effectively. It provides tools for monitoring financial activities, planning budgets, and maintaining better control over spending habits.


## ✨ Features

### 📍 Core Features
- **Add Income** - Record income and assign it to a specific savings account
- **Add Expense** - Record expenses and deduct from a specific savings account
- **Transfer Money** - Move funds between your personal savings accounts.

### 📈 Analytics & Summary
- **Dashboard** - Overview of all savings balances and overall total.
- **Monthly Bar Graph** - Visualize total expenses per month by year.
- **Monthly Pie Chart** - Compare income vs expenses for the current month.

### 🔜 Future Planning
- **Future Income & Expenses** - Plan upcoming transactions without affecting current balance.
- **Projected Balance** - See estimated balance per savings based on future plans.

### 📝 Reports
- **Save as PDF** - Export income records filtered by month or all time.

### ➕ Other
- **Loan Tracker** - View all expenses tagged under the "Loan" category.
- **Recent Transactions** - Quick view of latest income and expenses on dashboard.
- **Secure Login** - Password-only authentication with JWT tokens.
- **Filter by Month** - Filter income and expense tables by month.
